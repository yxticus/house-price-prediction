# House Price Prediction

## Overview
This project uses the California Housing dataset to analyze the factors that influence house prices and build a regression model to predict property values.

## What I Did
- Loaded and explored a real-world housing dataset
- Analyzed summary statistics and feature relationships
- Visualized correlation between features and house price
- Built a Linear Regression model
- Split the data into training and testing sets
- Evaluated model performance using Mean Squared Error (MSE) and R²
- Interpreted model coefficients to understand which features influence price most

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- scikit-learn

## Key Insights
- Some features show stronger correlation with house price than others
- Regression can be used to estimate housing values from multiple variables
- Model coefficients help explain the direction and strength of feature impact
